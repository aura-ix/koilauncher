# KoiLauncher

Fork of [Olauncher](https://github.com/tanujnotes/Olauncher). GNU GPLv3.

Changes from Olauncher:

- Navbar is now hidden, but accessing the recent apps view can be done by tapping the wallpaper.
- Defaults to following the system theme, instead of a dark theme.
- Removed all nag dialogues.
- Support for using volume buttons as page up/page down in the app drawer for use on ereaders.
